# Wireflows iniciais

## WF-01 Descoberta local

1. Usuário abre home.
2. Concede localização ou define ponto manual.
3. Aplica filtros (categoria, distância, avaliação).
4. Visualiza lista + mapa sincronizados.
5. Acessa detalhe do local e inicia contato.

## WF-02 Prestador de serviço

1. Prestador cria conta.
2. Completa perfil profissional e região atendida.
3. Envia documentação para credenciamento.
4. Recebe status (pendente/aprovado/rejeitado).
5. Passa a aparecer na busca.

## WF-03 Solicitação de reserva de imóvel

1. Visitante consulta imóvel.
2. Seleciona período e envia solicitação.
3. Host aprova ou recusa.
4. Sistema registra status e notifica as partes.

## WF-04 Moderação admin

1. Operação acessa fila de cadastros pendentes.
2. Analisa documentos e dados.
3. Aprova/reprova com motivo.
4. Evento é auditado e usuário notificado.
